- 策略模式
	- 表单字段
	- 权限验证
- 发布订阅模式
- 装饰器模式
	- 为一个旧对象，新增方法，返回装饰器对象
- 适配器模式
	- **将不同的数据结构，不同来源的数据适配成展示组件所能接受的数据结构**
- 代理模式
	- 为其它对象提供一种代理以**控制**这个对象的访问，具体执行的功能还是这个对象本身
		- 我们发邮件，通过代理模式，那么代理者可以控制，决定发还是不发，但具体发的执行功能，是外部对象所决定，而不是代理者决定
- 责任链模式
	- ```
	  // 运用责任链模式实现上边功能
	  chainApplyLincense.setNext(chainSelectChecker);
	  chainApplyLincense.run();
	  ```
