- 使用ES6的set关键字
	- ```
	  function unique(arr){
	  	return [...new Set(arr)];
	  }
	  ```
- 使用ES5的[[filter]]方法
	- filter根据函数参数进行筛选，并返回一个新数组
	- [[indexOf]]方法返回第一个出现的位置
	- ```
	  function unique(arr){
	  	return arr.filter(item,index,array) =>{
	      		return array.indexOf(item) === index;
	      }
	  }
	  ```
